# Wiki Link Bot для Telegram-группы

Бот читает сообщения в группе, пытается найти релевантную страницу в вашей вики (по `sitemap.xml`) и, если совпадение достаточно хорошее, отвечает ссылкой на страницу.

## Что нужно

- Python 3.11+ (желательно)
- Токен Telegram-бота из BotFather

## Быстрый старт

1) Установить зависимости:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2) Создать `.env` рядом с `requirements.txt` (можно скопировать из примера):

```bash
copy .env.example .env
notepad .env
```

Важно: файл `.env` **не коммитьте** — он добавлен в `.gitignore`. В репозиторий коммитится только `.env.example`.

Минимально нужно заполнить:

- `TELEGRAM_BOT_TOKEN`
- `WIKI_BASE_URL`
- `WIKI_SITEMAP_URL`

3) Запустить:

```bash
python -m app.bot
```

Или (Windows) открыть отдельное окно с логами:

```bat
start-bot.cmd
```

Остановить (если запускали через `start-bot.cmd`):

```bat
stop-bot.cmd
```

Перезапустить:

```bat
restart-bot.cmd
```

4) Добавить бота в группу и написать в группе команду:

`/id`

Бот ответит числовым `chat_id`. Если хотите ограничить бота только этой группой — впишите это значение в `.env` как `ALLOWED_CHAT_ID`.

## Как это работает

- Из `WIKI_SITEMAP_URL` берётся список страниц
- Страницы скачиваются и складываются в кэш `CACHE_PATH`
- Для каждой страницы строится “текст документа” (title + текст страницы без скриптов/стилей)
- Для каждого сообщения в чате считается похожесть (RapidFuzz `token_set_ratio`)
- Если лучший результат \(\ge\) `MIN_SCORE`, бот отвечает ссылкой

## Настройки

Смотрите `.env.example`.

Полезное:

- `MIN_SCORE`: порог (0..100). Если слишком много ложных — поднимите.
- `QUESTIONS_ONLY=true`: отвечать только на вопросы ("?" или вопросительные слова).
- `ALLOWED_CHAT_ID`: ограничить работу только одной группой (ID чата).
- `WIKI_MAX_PAGES`: ограничить количество страниц для индекса (чтобы первый запуск был быстрее).
