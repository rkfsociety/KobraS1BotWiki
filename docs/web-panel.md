# Веб-панель администратора

Встроенная панель (`app/web_panel.py`) запускается в фоновом потоке внутри процесса бота — отдельный сервис не нужен. Включается при `PANEL_ENABLED=true` или автоматически, если задан `PANEL_PASSWORD`.

## Возможности

- **Дашборд** — лента последних ответов бота с кнопкой «Отметить ошибочным» (⚑), статус, размер индекса, ключевые параметры
- **Ошибочные ответы** — список помеченных ответов (`data/bad_answers.json`); кнопка «Обработано» удаляет запись
- **Ручные Q&A** — просмотр / добавление / редактирование / удаление `data/manual_qa.json` (аналог `/qaadd`, `/qalist`, `/qadel`); правки сразу применяются без перезапуска
- **Фиксы ссылок** — управление переопределениями (`fixes.json`, аналог `/fix`)
- **Логи** — хвост `logs/bot.log` с фильтром по подстроке
- **Обновление** — «Проверить обновления» (git fetch) и «Обновить» (git pull + перезапуск)
- **Настройки** — редактирование `.env` прямо из браузера (кроме токена бота)

## Вход

### Через Telegram-бота (рекомендуется)

Не требует домена — работает на «голом» IP:порт.

1. На странице входа нажать «Войти через Telegram-бота»
2. Откроется `t.me/<бот>?start=<код>` — нажать **Start**
3. Бот через `get_chat_member` проверяет, что пользователь — **админ группы** `PANEL_ADMIN_CHAT_ID`
4. Код одноразовый, живёт 5 минут, привязан к браузеру

### По логину/паролю (запасной)

Активен только если задан `PANEL_PASSWORD`. Членство в группе не проверяется.

### Login Widget (опционально)

Endpoint `/tg-auth` поддерживает классическую кнопку «Log in with Telegram». Требует домен — в @BotFather → `/setdomain`. На «голом» IP не работает.

## Настройки панели

| Переменная | По умолчанию | Описание |
|-----------|-------------|---------|
| `PANEL_ENABLED` | `true` если задан пароль | Включить панель. Для TG-входа без пароля задайте явно `true`. |
| `PANEL_TG_LOGIN` | `true` | Вход через Telegram. |
| `PANEL_ADMIN_CHAT_ID` | `-1003881305021` | Группа, чьи админы могут входить. `0`/`off` — выключить TG-вход. |
| `PANEL_PASSWORD` | — | Пароль запасного входа. Пусто = только через Telegram. |
| `PANEL_USERNAME` | `admin` | Логин запасного входа. |
| `PANEL_HOST` | `0.0.0.0` | Адрес прослушивания. `127.0.0.1` — только локально. |
| `PANEL_PORT` | `8080` | Порт. |
| `PANEL_SESSION_TTL_SECONDS` | `86400` | Срок жизни сессии (секунды). |

## Безопасность

- TG-вход: проверка HMAC-подписи + членство в админах группы
- Сессия: cookie (`HttpOnly`, `SameSite=Lax`)
- Формы: CSRF-токен
- Вход: троттлинг (8 неудач за 5 минут → пауза)
- Панель работает по **HTTP** — при доступе снаружи закройте reverse-proxy с HTTPS (nginx/Caddy); это заодно даёт домен для `/setdomain`
